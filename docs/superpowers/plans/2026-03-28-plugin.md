# CodeGuard Plugin Module Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the CodeGuard Claude Code plugin that bundles a ReviewAgent, review skills, hooks, a slash command, and an upload script — all as markdown, JSON, and shell files requiring no compilation.

**Architecture:** A Claude Code plugin directory (`codeguard-plugin/`) with a manifest in `.claude-plugin/plugin.json`, a ReviewAgent sub-agent definition, four review skills (general, security, performance, output-format), a `Stop` hook to auto-trigger reviews, a `SubagentStop` hook to upload results, a `/codeguard:review` slash command for manual triggers, and a settings file. The plugin uses only markdown, JSON, and shell — no build step.

**Tech Stack:** Claude Code Plugin SDK (markdown agents, skills, hooks, commands), JSON configuration, Bash (upload script), Git.
