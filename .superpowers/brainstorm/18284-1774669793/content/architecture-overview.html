关键设计决策

Plugin

标准 Claude Code 插件格式，Hook 驱动自动触发，Skills 按需加载

Server

FastAPI 异步服务，存储层抽象（SQLite 默认 / PostgreSQL 可选），API Key 认证

Web UI

React + TS + Vite，按项目组织 Review，支持 evidence chain 展示

部署

Docker Compose 两容器（Server + Nginx），一键启动
